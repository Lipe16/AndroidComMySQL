# AndroidComMySQL
Projeto de android com Mysql utilizando PHP e Json objects


![App](https://image.ibb.co/h00PDS/biblioteca_ION.png)
##Ao enves decarregar uma página na WEB, você pode carregar uma página rodando localmente também...

Este projeto foi criado para mostrar como trabalhar com banco de dados usando o Android, nele mostro que não devemos conectar o dispositivo android direto no MYSQL mais sim fazer o dispositivo android consumir um serviço que faça isso por ele.
O serviço pode ser visto da seguinte maneira: uma página PHP que reconheça parametros enviados por GET ou POST, e utilize esses parametros para inserir dados no BD ou atualizar, deletar...
